Project Description:

This Car Sharing App allows users from around the world to find and share car trips conveniently. The app provides an intuitive interface for drivers to post available seats in their vehicles, and for riders to book these seats, optimizing travel costs and reducing environmental impact.

Features:

  • User Profiles: Users can create accounts to post or join rides.
  • Real-time Updates: Rides are updated in real-time using AJAX and JSON to provide the most current listings without page reloads.
  • Search and Filter: Advanced search options enable users to find rides based on location, date, and preferences.


Technologies Used:
